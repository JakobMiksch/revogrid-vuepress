# Grouping(Stacked columns)

Column can be defined as group or as regular column. To stack column just define it as a group. Check interfaces for more information.


const columns = [
  { name: 'Name', prop: 'name', }, // regular column

  // stacked column
  {
      name: 'Personal',
      children: [
          { name: 'Eyes', prop: 'eyeColor', }, prop: 'age', }, // regular column
          { name: 'Company', prop: 'company' }, // regular column
      ]
  }
];
